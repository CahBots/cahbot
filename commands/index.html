CahBot Commands

Command List

Command Description
Typing "CB Prefix" Reminds you of the prefix.
^info Shows you some info about CB, or something.
^rnumber [Num] [Higher Num] Gives you a random number between a given range.
^help Basically tells you to go here
^cmds Pulls up the command list
^eightball Ask the 8ball something
^userinfo Shows some info about you.
^reverse Reverses text.
^flip Flips a coin, what else did you expect?
^flop Flops a coin, what expect did you else??
^ping Used to show response time
^servercount Returns the number of servers CB is in
^invite Gives you a link to invite me to your own server!
^die Shuts CahBot down, only Cah can use this command
^roll Rolls a number between 1 and 6
^eval Like you don't know what eval commands do
^donate Want to donate? That's great! This command gives you a link for Donorbox donationso
^update Gives you the latest CB update
^say Makes CB say something, you need the manage messages perm tho
^thanks Thanks to these radical donors!
^feedback [words] Want to tell me my bot sucks without going into the CB Server? Well, here you go.
^noot noot
